What is this?

A SMITE 2 Conquest build helper. You choose the role you queue, open a god, and get a buy order made for that god’s abilities — with a short “why” under items. Also: counter paths for the enemy team, and meme troll builds.

What this website is for

          

            SMITE 2 Conquest build helper.
            Open a role, pick a god, and follow a numbered buy order built around
            that god’s kit and recent patch signals — not a one-size-fits-all pro site copy.

How builds are weighted

          

            Nothing here is live win rate. Paths come from wiki stats, ability text, and
            patch-note impact we score locally — not from pro sites.
          

          

            

              
Two different systems

              
    
                
  • Builds answer: “What should this god buy in this role?”
    • 
                
  • Tiers answer: “Who looks strong this patch overall?”
    • 
              

              

                A god can be high tier and still have a different buy order than another god in the same role.
              

            

            

              
Build pipeline (god-first)

              
    
                
  1. Role score — each role has stat weights (e.g. Carry: AS/crit/pen; Support: prots/auras/anti-crit). Items also get utility, cost, active tax, and item patch momentum.
    2. 
                
  2. God re-score — role score is reduced (~55%), then kit affinity is added hard: STR vs INT, physical vs magical, tags (DoT, AA, heal, CC…), kit effects, and that god’s recent patch axes.
    3. 
                
  3. Assemble a path — not “global top 6.” We fill archetype slots (power / pen / CDR / defense…), diversify so gods don’t clone each other, inject signature items, cap luxury actives (usually ≤2 shop actives), and enforce pen on damage roles.
    4. 
                
  4. Explain — path “why” + per-item “Why this?” from the signals that picked the item.
    5. 
              

            

            

              
Tiers (separate ladder)

              
    
                
  • Patch ~48% — note momentum (recent patches count more)
    • 
                
  • Kit ~28% — how strong/flexible the kit metrics look
    • 
                
  • Build fit ~14% — how well the Conquest engine can itemize them
    • 
                
  • Novelty / stability ~5% each
    • 
              

              

                S/A/B/C/D are percentiles inside a scope (overall, role, damage type) — not “pro letter grades.”
                Tier letters on build cards are ranking context, not “buy this because S-tier.”
              

            

            

              
Counter & Troll

              
    
                
  • Counter — starts from the kit path, then re-weights for the enemy 5 (magic/phys mix, crit ADC, healing, CC).
    • 
                
  • Troll — kit-true off-meta paths for frustration/fun. Not ranked advice.
    • 
                
  • Aspects — same pipeline, different kit input (separate base vs aspect paths when available).
    • 
              

            

          


          
Role jobs

          
    
            
  • Carry / Mid — backline damage + pen; Support peels.
    • 
            
  • Jungle — ganks (Bumba, pen, CDR, Blink).
    • 
            
  • Solo — unkillable frontline (prots, HP, Damp/Plat/Ten).
    • 
            
  • Support — peel for ADC/mid (mitigate, anti-AS/crit).
