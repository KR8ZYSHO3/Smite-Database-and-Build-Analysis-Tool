What this website is for

          

            SMITE 2 Conquest build helper.
            Open a role, pick a god, and follow a numbered buy order built around
            that god’s kit — not a one-size-fits-all pro site copy.

How builds are weighted

          

            Nothing here is live win rate. Paths come from wiki stats, ability text, and
            patch-note impact — not from pro sites.

  1. Role score — each role has stat priorities (e.g. Carry wants AS/crit/pen).

  2. Item tier ladder — S/A items from the item tier list get a real boost; C/D get soft penalties (role-gated so tank S-tiers don’t flood Mid).
    3. 
                
  3. God re-score — kit tags and abilities change the ranking hard.
    4. 
                
  4. Assemble — fill a 6-item path with pen rules and active caps.
