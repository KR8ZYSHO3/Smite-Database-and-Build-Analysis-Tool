About this tool

          

            Local / GitHub Pages viewer for SMITE 2 only data
            (gods, abilities, items, patch notes). Not Smite 1.
          

          
    
            
  • Tier lists — patch-note momentum (48%) + kit (28%) + build fit (14%) + novelty/stability.
    • 
            
  • Conquest builds — 1 starter + 6 items in buy order. Default 2 shop actives (game hard max 3; curio shares that budget). Damage roles require real pen.
    • 
            
  • Source — official wiki.smite2.com via the Python scraper in this repo.
    • 
          

          

            This is a statistical model, not live win rates. Use Scope = overall for the full S-tier list;
            role scopes only promote ~top 12% of that role (~3 S gods).
